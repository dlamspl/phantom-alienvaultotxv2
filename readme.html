The app provides two main features.
  • It can be used ad-hoc or in playbooks to query OTX for URL, IP and hash reputation.
  • Can be used as ingest source. In this case it will retrieve the last 24 hours new pulses and extract relevant artifacts
In the case where it is used as ingest source the following artifacts are extracted/supported currently. These are OTXv2 specific names and are transformed to the appropriate CEF name and type for phantom.
  • FileHash-MD5
  • FileHash-SHA256
  • FileHash-SHA1
  • domain
  • IPv4
  • URL
  • hostname
  • email
  • CIDR
